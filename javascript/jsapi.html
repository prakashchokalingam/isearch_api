Register :

In order get access of our database you need to register in our developer console !

                    (i)  Provide essential details which are asked !

        (ii) Save the auth_id and secret_id !

  (iii) Keep  the secret_id safe!

Download :

You have choosen to download the Javascript (jquery ) SDK
Jquery plugin is must : 

                 <script src="http://ajax.googleapis.com/ajax/
libs/jquery/1.11.1/jquery.min.js"></script>


Authentication :

Need auth_id and secret id to get auth_token

Input codes :


  <script type="text/javascript" src="isearch.js"></script>
      <script type="text/javascript">

   // declare the auth_id and secret_id varible 
    // and call the function
     var json=authenticate(auth_id,secret_id);
   //output will be return as json
     </script>

Output

if success : 
{"auth_token":"xxxxxxxxxxxxxxx"}
else 
{"error":"Check your input parameters !"}
